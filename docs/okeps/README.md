# OVN-Kubernetes MCP — Enhancement Proposals (OKEPs)

Design proposals for the `ovn-kubernetes-mcp` server and the
troubleshooting workflows it enables. Format follows the
[upstream ovn-kubernetes OKEP process](https://ovn-kubernetes.io/okeps/):
each OKEP is filed against a tracking issue and named
`okep-<issue-number>-<short-title>.md`.
