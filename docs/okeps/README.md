# OVN-Kubernetes MCP — Enhancement Proposals (OKEPs)

OKEPs in this directory describe design decisions for the
`ovn-kubernetes-mcp` server and the troubleshooting workflows it enables.

The format mirrors the upstream
[ovn-kubernetes OKEP process](https://github.com/ovn-kubernetes/ovn-kubernetes/tree/master/docs/okeps):
each OKEP is filed against a tracking issue and named
`okep-<issue-number>-<short-title>.md`.
